# OSIS Java Projects

This repository contains Java console-based applications developed as part of my Java programming practice.

## Projects Included

### Task 1 – Reservation System
A railway reservation system developed using Java.

**Features**
- Reserve Ticket
- Cancel Ticket
- View Reservation Details

---

### Task 2 – Number Guessing Game
A simple game where the user guesses a randomly generated number.

**Features**
- Random Number Generation
- Score Calculation
- Multiple Attempts

---

### Task 3 – ATM Interface
A console-based ATM simulation.

**Features**
- Deposit Money
- Withdraw Money
- Check Balance
- Transaction History

## Technologies Used

- Scanner Class
